# 기여 가이드라인 (Contributing Guidelines)

기술 블로그 모음 저장소에 관심을 가져주셔서 감사합니다!  
이 프로젝트는 여러분의 참여로 유지됩니다. 아래 규칙을 확인하신 후 편하게 Pull Request를 보내주세요.

## 기여 방법 (Step by Step)

1. 이 저장소를 **Fork**한 뒤, 본인의 컴퓨터로 clone 합니다.
2. `README.md` 파일에서 알맞은 카테고리(국내/해외) 표에 항목을 추가합니다.
    - **비고란 작성 규칙:** 만약 추가하려는 블로그가 최근 1년 이상 업데이트가 없거나 폐쇄된 곳이라면 비고 열에 `🔴` 표시를 함께 달아주세요.
    - 예시: `| 기업명 입력 | 블로그 링크 입력 | 비고 입력 |`
3. 수정이 완료되면 커밋 메시지를 규칙에 맞게 작성하여 본인 레포에 Push 합니다.
    - 예시: `docs: 우아한형제들 기술 블로그 추가`
4. 본인 브랜치에서 이 저장소의 `main` 브랜치로 **Pull Request**를 생성합니다.
