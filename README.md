> **국내외 기업들의 엔지니어링 기록이 담긴 '기술 블로그 모음' 저장소**

스타트업부터 글로벌 테크 기업까지 다양한 규모의 국내외 기업 기술 블로그 모음을 한곳에서 편리하게 찾아볼 수 있습니다.

이 저장소는 여러분과 함께 완성해 나가는 공간입니다. 나만 알기 아까운 보석 같은 회사 기술 블로그가 있거나 주소가 바뀐 곳을 발견하셨다면, 언제든 편하게 Pull Request를 보내주세요. 양질의 기술 아티클과 엔지니어링 기록을 통해 함께 성장해요!
